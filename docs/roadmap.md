# Roadmap Técnico

## Fase 1 (entregue nesta base)
- Fundamentos da API e autenticação
- Estrutura de pastas conforme arquitetura alvo
- RBAC básico e escopo de OU
- Dashboard e listagem de usuários (modo demo)
- Auditoria e hardening inicial

## Fase 2
- Integração LDAP completa (consulta e paginação real)
- Operações de usuários/computadores/grupos/OUs com segurança transacional
- WebSocket com eventos reais de operação
- Exportação CSV/XLSX/PDF para logs

## Fase 3
- Instalação one-shot por distribuição Linux
- CI/CD com lint/test/build/scan
- Modo plugin
- UI avançada ADUC-like com árvore drag-and-drop
